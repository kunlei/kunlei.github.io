C++ - string

 Posted on January 14, 2020  |  2 minutes  |  232 words  |  Kunlei Lian
    • Creation
    • Access
    • Capacity
    • getline()

The string is a library class defined by C++ to represent and manipulate characters. It’s defined in header string and included in namespace std:

1#include <string>
2using std::string;

We’ll look at some of the most useful operations for string.

Creation

There exist a number of constructors for string.

1string(); // default constructor
2string(const string &str); // copy constructor
3string(const char * c); // c-string constructor
4string(size_t n, char c); // fill constructor

1string a; // define empty string using default constructor
2string b(a); // copy construction
3string c("hello world"); // c-string constructor
4string d(10, d); // fill construction

Access

The library defines a number of iterators to iterate through a string object.

1string s("hello word");
2for (auto itr = s.begin(); itr != s.end(); ++itr) {
3  cout << *itr << endl;
4}

Elements in a string can also be accessed directly.

1string s(10, 'a');
2cout << s[1] << endl;
3cout << s.at(2) << endl;

Other available iterators include rbegin(), rend(), cbegin(), cend(), crbegin() and crend().

Capacity

We can use functions to inquiry the capacity of a string.

1string s("string");
2cout << s.size() << endl;
3cout << s.empty() << endl;

getline()

getline() is a useful function to retrieve string from input stream.

1std::istream& getline(std::istream& istrm, string& str);

For example, the following code reads one line from the input stream and saves it into a string.

1string s;
2getline(cin, s);
3cout << s << endl;
