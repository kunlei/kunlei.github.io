Recipe - Plotting Multiple Histograms in one Graph

 Posted on July 23, 2020  |  1 minutes  |  120 words  |  Kunlei Lian

I recently had a small data analysis work that required showing two histograms in the same graph. It has been quite a while since my I last used R in my work, and I had to resort to google to find the solution (see the code below).

 1# create two sample data frames
 2x <- data.frame(x = sample(0 : 10, 100, TRUE), group = 'Group A')
 3y <- data.frame(x = sample(5 : 20, 50, TRUE), group = 'Group B')
 4
 5# bind them together
 6data <- rbind(x, y)
 7
 8# make the histogram using ggplot
 9ggplot(data, aes(x, fill = group, colour = group)) +
10  geom_histogram(aes(y = ..density..), breaks=seq(0, 20, 1), alpha=0.6, 
11                 position="identity", lwd=0.2)
