Capacitated Facility Location Problem - MIP Model Solving using CPLEX

 Posted on July 19, 2020  |  10 minutes  |  1943 words  |  Kunlei Lian
    • Testing instance
    • Model building and solving

I’ll demonstrate in this post how to use the CPLEX Java API to solve an capacitated facility location problem instance. A testing instance cap41 is taken from ORLib and the data input function will be illustrated first. With the class holding the instance data ready, the MIP modeling class will be implemented and computational results will be shown in the end.

Testing instance

The instance can be obtained directly from the following url:

1http://people.brunel.ac.uk/~mastjjb/jeb/orlib/files/cap41.txt

I’ll use the Instance class to hold all the data, which encompasses some key information of this problem:

  • facility-related data
    • numFacilities: number of facilities available
    • fixedCosts: fixed cost of each facility
    • capacities: capacity of each facility
  • customer-related data
    • numCustomers: number of customers
    • demands: demand of each customer
  • assignment-related data
    • flowCosts: cost of serving a customer from a specific facility

The code below gives the Instance class definition, the key part of which is the readData function that takes an filename as input and scans all the data within the file.

 12   * total number of facilities
 13   */
 14  private int numFacilities;
 15
 16  /**
 17   * total number of customers
 18   */
 19  private int numCustomers;
 20
 21  /**
 22   * facility capacities
 23   * size: 1 * numFacilities
 24   */
 25  private int[] capacities;
 26
 27  /**
 28   * facility fixed costs
 29   * size: 1 * numFacilities
 30   */
 31  private double[] fixedCosts;
 32
 33  /**
 34   * customer demands
 35   * size: 1 * numCustomers
 36   */
 37  private int[] demands;
 38
 39  /**
 40   * allocation cost from facilities to customers
 41   * size: numCustomers * numFacilities
 42   */
 43  private double[][] flowCosts;
 44
 45  public Instance() {
 46    numCustomers = 0;
 47    numFacilities = 0;
 48    capacities = null;
 49    fixedCosts = null;
 50    demands = null;
 51    flowCosts = null;
 52  }
 53
 54  /**
 55   * read data from file
 56   * @param filename name of the input file
 57   */
 58  public void readData(String filename) {
 59    try {
 60      File file = new File(filename);
 61      Scanner sc = new Scanner(file);
 62
 63      numFacilities = sc.nextInt();
 64      numCustomers = sc.nextInt();
 65
 66      // create data objects
 67      capacities = new int[numFacilities];
 68      fixedCosts = new double[numFacilities];
 69      demands = new int[numCustomers];
 70      flowCosts = new double[numCustomers][numFacilities];
 71      for (int i = 0; i < numCustomers; ++i) {
 72        flowCosts[i] = new double[numFacilities];
 73      }
 74
 75      // read in facility capacities and fixed cost
 76      for (int f = 0; f < numFacilities; ++f) {
 77        capacities[f] = sc.nextInt();
 78        fixedCosts[f] = sc.nextDouble();
 79      }
 80
 81      // read in customer demand and flow cost
 82      for (int c = 0; c < numCustomers; ++c) {
 83        demands[c] = sc.nextInt();
 84        for (int f = 0; f < numFacilities; ++f) {
 85          flowCosts[c][f] = sc.nextDouble() / demands[c];
 86        }
 87      }
 88
 89      sc.close();
 90    } catch (FileNotFoundException e) {
 91      e.printStackTrace();
 92    }
 93  }

Model building and solving

The modeling code is pretty self-explanatory. The cplex object is first created within the constructor to help us build key modeling components:

  • decision variables
    • open: a binary variable equals 1 if a facility is open, 0 otherwise
    • flow: a continuous variable that represents the demand of a customer that’s satisfied by a facility
  • objective
    • obj: this defines the objective function that consists of the fixed cost and assignment cost
  • constraints
    • consDemand: the constraint requires that all customer demands must be fulfilled
    • constCapacity: this constraint makes sure that facility capacity is not violated

The solve() function calls CPLEX to find and output the optimal solution.

  1package com.learn.opt.benders;
  2
  3import ilog.concert.IloException;
  4import ilog.concert.IloLinearNumExpr;
  5import ilog.concert.IloNumVar;
  6import ilog.concert.IloObjective;
  7import ilog.concert.IloRange;
  8import ilog.cplex.IloCplex;
  9
 10public class MIPModel {
 11  private final Instance instance;
 12
 13  private IloCplex cplex;
 14
 15  private IloNumVar[] open;
 16  private IloNumVar[][] flow;
 17
 18  private IloObjective obj;
 19  private IloRange[] consDemand;
 20  private IloRange[] consCapacity;
 21
 22  public MIPModel(Instance instance) {
 23    this.instance = instance;
 24
 25    int numFacilities = instance.getNumFacilities();
 26    int numCustomers = instance.getNumCustomers();
 27
 28    try {
 29      cplex = new IloCplex();
 30
 31      // create decision variables that indicate whether a facility is open or not
 32      open = new IloNumVar[numFacilities];
 33      for (int f = 0; f < numFacilities; ++f) {
 34        open[f] = cplex.boolVar("open_" + f);
 35      }
 36
 37      // create decision variables that represent the demand of customer c that are
 38      // satisfied by facility f
 39      flow = new IloNumVar[numFacilities][numCustomers];
 40      for (int f = 0; f < numFacilities; ++f) {
 41        for (int c = 0; c < numCustomers; ++c) {
 42          flow[f][c] = cplex.numVar(0, Double.MAX_VALUE, "flow_" + f + "," + c);
 43        }
 44      }
 45
 46      // create objective function
 47      IloLinearNumExpr expr = cplex.linearNumExpr();
 48      for (int f = 0; f < numFacilities; ++f) {
 49        expr.addTerm(instance.getFixedCost(f), open[f]);
 50        for (int c = 0; c < numCustomers; ++c) {
 51          expr.addTerm(instance.getFlowCost(f, c), flow[f][c]);
 52        }
 53      }
 54      obj = cplex.addMinimize(expr, "obj");
 55
 56      // create constraints
 57      consDemand = new IloRange[numCustomers];
 58      for (int c = 0; c < numCustomers; ++c) {
 59        expr.clear();
 60        for (int f = 0; f < numFacilities; ++f) {
 61          expr.addTerm(1.0, flow[f][c]);
 62        }
 63        consDemand[c] = cplex.addGe(expr, instance.getDemand(c), "demand_" + c);
 64      }
 65
 66      consCapacity = new IloRange[numFacilities];
 67      for (int f = 0; f < numFacilities; ++f) {
 68        expr.clear();
 69        for (int c = 0; c < numCustomers; ++c) {
 70          expr.addTerm(1.0, flow[f][c]);
 71        }
 72        expr.addTerm(-instance.getCapacity(f), open[f]);
 73        consCapacity[f] = cplex.addLe(expr, 0, "capacity_" + f);
 74      }
 75
 76    } catch (IloException e) {
 77      e.printStackTrace();
 78    }
 79  }
 80
 81  public void solve() {
 82    try {
 83      if (cplex.solve()) {
 84        System.out.println("obj = " + cplex.getObjValue());
 85
 86        int numFacilities = instance.getNumFacilities();
 87        int numCustomers = instance.getNumCustomers();
 88        double[] optOpen = cplex.getValues(open);
 89        double[][] optFlow = new double[numFacilities][];
 90        for (int f = 0; f < numFacilities; ++f) {
 91          optFlow[f] = cplex.getValues(flow[f]);
 92        }
 93
 94        for (int f = 0; f < numFacilities; ++f) {
 95          System.out.print("facility " + f + ": ");
 96          if (optOpen[f] > 0.99) {
 97            System.out.println("open, ");
 98            for (int c = 0; c < numCustomers; ++c) {
 99              if (optFlow[f][c] > 0.01) {
100                System.out.println("\tto customer " + c + " volume: " + optFlow[f][c]);
101              }
102            }
103          } else {
104            System.out.println("close, ");
105          }
106        }
107      }
108    } catch (IloException e) {
109      e.printStackTrace();
110    }
111  }
112
113}
