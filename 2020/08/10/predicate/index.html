Functional Interface - Predicate

 Posted on August 10, 2020  |  3 minutes  |  450 words  |  Kunlei Lian

Functional interface in Java is a special type of interface that has only one abstract method. A typical example is Predicate<T> which provides a test(T t) method to return a boolean variable:

1@FunctionalInterface
2public interface Predicate<T> {
3  boolean test(T t);
4}

Note that @FunctionalInterface is not required here but can help the JDK recognize our intention. Classes implementing this interface provide an evaluation function to decide whether a statement about a T object is true or not. This is often used in filtering functions where we establish a filter criterion to decide whether an object meet the requirement or not.

To illustrate this, let’s define a Car class with three attributes, namely, color, weight and year:

 1package com.learn.java.java8demo;
 2
 3public class Car {
 4  private String color;
 5  private double weight;
 6  private int year;
 7
 8  public Car(String color, double weight, int year) {
 9    this.color = color;
10    this.weight = weight;
11    this.year = year;
12  }
13
14  public String getColor() {
15    return color;
16  }
17
18  public double getWeight() {
19    return weight;
20  }
21
22  public int getYear() {
23    return year;
24  }
25
26  @Override
27  public String toString() {
28    return "Car{" +
29        "color='" + color + '\'' +
30        ", weight=" + weight +
31        ", year=" + year +
32        '}';
33  }
34}

Now we have a collection of cars:

1List<Car> cars = new ArrayList<>();
2cars.add(new Car("green", 1000.0, 2018));
3cars.add(new Car("blue", 2000.0, 2020));
4cars.add(new Car("maroon", 3000.0, 2019));
5cars.add(new Car("yellow", 2500.0, 2020));
6cars.add(new Car("purple", 999.0, 2021));

If we want to view all cars with weight bigger than 1000 pounds, we can define a implementation class:

1import java.util.function.Predicate;
2
3public class CarWeightPredicate implements Predicate<Car> {
4  @Override
5  public boolean test(Car car) {
6    return car.getWeight() > 1000.0;
7  }
8}

This predicate can then be supplied into a filter function:

1Predicate<Car> weightPredicate = new CarWeightPredicate();
2cars.stream().filter(weightPredicate)
3    .collect(Collectors.toList())
4    .forEach(System.out::println);
5System.out.println();
Car{color='blue', weight=2000.0, year=2020}
Car{color='maroon', weight=3000.0, year=2019}
Car{color='yellow', weight=2500.0, year=2020}

A less verbose way to define a predicate is with the help of lambda function in Java 8:

1Predicate<Car> yearPredicate = car -> car.getYear() > 2019;
2cars.stream().filter(yearPredicate)
3    .collect(Collectors.toList())
4    .forEach(System.out::println);
5System.out.println();
Car{color='blue', weight=2000.0, year=2020}
Car{color='yellow', weight=2500.0, year=2020}
Car{color='purple', weight=999.0, year=2021}

Composite predicates can be defined by combining multiple predicates together:

 1Predicate<Car> andPredicate = weightPredicate.and(yearPredicate);
 2cars.stream().filter(andPredicate)
 3    .collect(Collectors.toList())
 4    .forEach(System.out::println);
 5System.out.println();
 6
 7Predicate<Car> orPredicate = weightPredicate.or(yearPredicate);
 8cars.stream().filter(orPredicate)
 9    .collect(Collectors.toList())
10    .forEach(System.out::println);
11System.out.println();
12
13Predicate<Car> negatePredicate = weightPredicate.negate();
14cars.stream().filter(negatePredicate)
15    .collect(Collectors.toList())
16    .forEach(System.out::println);
